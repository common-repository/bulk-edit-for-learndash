== Description ==
Bulk Edit for Learndash is the best way to bulk edit courses prices.

Main features:

* Define a price and apply it for all courses with a single click
* Edit courses prices by LD course category


== Installation ==
1. Upload plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Visit settings through "Grid Button" (LearnDash - LMS submenu) link.
